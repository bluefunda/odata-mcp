# Cursor example

Add the MCP server to Cursor via **Settings → Cursor Settings → MCP → Add new global MCP server**.

Or create / edit `.cursor/mcp.json` in your project root (project-scoped) or
`~/.cursor/mcp.json` (global):

```json
// see mcp.json in this directory
```

Replace the placeholder values with your actual metadata path and credentials.

After saving, restart Cursor or run **MCP: Restart servers** from the command palette.
